Debug¶

Echo¶

returns the content, debug use

GET /api/echo¶

returns the query string as JSON

Note

all values are returned as arrays

/api/echo?a=b&a=c&c=d

gives:

{"a": ["b", "c"], "c": ["d"]}
POST /api/echo¶

returns JSON body POSTed

Status Codes:
  • 400 Bad Request – on invalid json
PUT /api/echo¶

returns JSON body PUT’d

Status Codes:
  • 400 Bad Request – on invalid json
